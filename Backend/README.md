## This is the backend part of Live Trading
## Firstly,You will clone the repository
## Now,We will create virtual environment for our project if .venv extension file is not present
## Run the command in your terminal
## python -m venv .venv
## Now,You have to enter into virtual environment. so you will run the command
## .venv/Scripts/activate

## Now,You will run the command
## pip install requirement.txt

## For install any library any project,run the command
## pip install library_name

## For run the project,Run command in terminal
## `uvicorn main:app --reload`
